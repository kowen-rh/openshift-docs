= Determining cluster access to the Machine API

Use the {oc-first} to determine if the cluster has access to the Machine API or not.

.Prerequisites

* Install the {oc-first}.

.Procedure

. Check if the Operator responsible for managing machines is running:
+
[source,terminal]
----
oc get clusteroperator machine-api
----
+
If the output contains `AVAILABLE=true`, you have the Machine API.
